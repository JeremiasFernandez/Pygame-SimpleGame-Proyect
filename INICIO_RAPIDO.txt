INICIO RAPIDO - Bossfight: El Troyano (Modo Desarrollo)
=======================================================

1) Instalar Python 3.10+ si no lo tienes.

2) Crear entorno virtual:
   Windows:
      python -m venv .venv
      call .venv\Scripts\activate.bat
   Linux/Mac:
      python3 -m venv .venv
      source .venv/bin/activate

3) Instalar dependencias:
      pip install -r requirements.txt

4) Ejecutar el juego:
      python Juego/Codigo/main.py

Scripts automáticos:
   Windows:  setup_env.bat
   Linux:    bash setup_env.sh

Estructura mínima:
  Juego/assets   -> recursos
  Juego/Codigo   -> código fuente (main.py)

Solución de problemas:
- ModuleNotFoundError: revisa que activaste el entorno y corriste pip install.
- FileNotFoundError assets: ejecuta siempre desde la raíz del repo (no dentro de Codigo).
- Pygame mixer error: verifica que tu sistema tenga salida de audio.

Opcional IA:
  PowerShell: $env:OPENAI_API_KEY="tu_api_key"

Fin.
