# Guía Rápida de Compilación
# ========================

## Para crear el ejecutable .exe:

### Método Automático (Recomendado):
1. Abre PowerShell en la carpeta Juego/Codigo
2. Ejecuta: .\build_exe.ps1
3. Espera a que termine (puede tomar 5-10 minutos)
4. El .exe estará en: dist/Bossfight_ElTroyano.exe

### Método Manual:
1. Instalar PyInstaller: pip install pyinstaller
2. Crear ícono: python create_icon.py
3. Compilar: pyinstaller --clean --noconfirm bossfight.spec
4. El .exe estará en: dist/Bossfight_ElTroyano.exe

## Archivos importantes:
- bossfight.spec: Configuración de PyInstaller
- create_icon.py: Generador de ícono para el .exe
- build_exe.ps1: Script automatizado de compilación

## Notas:
- El .exe incluye TODAS las dependencias
- No requiere Python instalado para ejecutarse
- Los assets están empaquetados dentro del .exe
- Tamaño aproximado: 50-100 MB
